
Freemath是以中文書寫的數學教材與、教法、學習方法的匯整
以開放源碼(open source)的授權釋出，您可以自由下載，以及截取其中的段落，融入自己的作品中

====使用方法====

許多教材需要列印成紙本使用，您可：

*在家裝置一台印表機
*使用其他空間的印表機
*使用便利商店的印表機
