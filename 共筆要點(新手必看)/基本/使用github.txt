共筆需知

===我可以任意修改專案的內容嗎？需不需要經過同意？===

創造自己的分叉，在自己的電腦修改，不必經過同意
提出修改的部份，要求整合進母專案，才需要經過同意

===該怎麼起步？===

請先上這裡註冊帳號
https://github.com

註冊成功之後
到這裡按右上的"Fork"
Fork就是分叉的意思，它會創造出你的專屬分叉
接著
到你的專屬分叉頁面，按右下的"Clone in Desktop"
就可以整個下載到電腦上跑

Windows使用說明：http://blog.sanc.idv.tw/2012/10/github-github.html

Mac下載點：http://mac.github.com

在電腦裡做了一些更動之後
使用commit + sync和網路上自己的分叉同步，記得寫上更動的筆記，讓別人看得懂
然後再按pull request => new pull request就可以向原本專案的協作者，提出整合的要求

===這流程看來對我有點複雜，有沒有更簡單的辦法？===

直接把你願意開源釋出的教材、文章、互動筆記、知識地圖…等等作品，po上臉書的<自學數學團>，並註明可供freemath計畫自由使用與改作。

===如何成為專案正式的協作者？===

正式協作者的更動不必經過申請，可以直接改母專案，權限很大。

因為雖然任何人都可以創建自己的專屬分叉，在分叉上任意改動，或在整個下載進電腦裡，自己任意改動；但是為了維持品質和理念，母專案本身並不像維基百科那樣，任何人都可以任意改動。

如果要成為專案的協作者，需要先申請Github帳號，並對專案有一定的貢獻，並參與合作討論，累積信用與聲望，再經過原本協作者的同意之後，成為正式的協作者。
