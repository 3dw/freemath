一元二次方程式的應用可說相當廣泛。

 

物理上，凡是和加速度有關的，像引力和彈道，都是二次式；經濟上，二次式可以分析最佳生產策略；幾何上，有名的黃金比例式就是二次的，圓方程也是二次的；數論上，它直接導致了無理數的發現。

 

「一元」的意思是，只有1個未知數。

「二次」的意思是，等式左右都是多項式[1]，其中最高次項是2次。

「方程式」的意思是：含未知數的等式。也就是說它有「=」號。

 

所以說，「x2 – 3x = 2x – 1」是一個【  】元【  】次【     】式。

「2x2 – 3x = 5」           是一個【  】元【  】次【     】式。

「-4x2 – x = 3」           是一個【  】元【  】次【     】式。

 

有的式子看起來不像，但是經過展開或去分母，就可以變成標準的一元二次方程式。它們也算是一元二次方程式。

 

練習：展開或去分母

如果你感到好奇，可以去看「極值問題」的單元。這裡只是一個簡介。

之前我們提過，十字交乘法不能分解所有的一元二次式。

如果解答是無理數時，十字交乘法就會配不出來。

 

像著名的黃金比例，是x2 - x - 1 = 0的解，

可以試試十字交乘，絕對做不出來。

但是，這表示x2 - x – 1不能因式分解嗎？

 

不，它還是可以。

 

〈這樣想想看〉

假如x2 - x - 1可以分解，必為(x – α)(x – β)的形式。[6]

而x2 - x - 1 = 0的解就是(x – α)(x – β) = 0的解，也就是【    】和【    】。

 

我們可以用配方法公式解，

輕易得到α =【         】, β=【          】。

 

所以代入得知(x – α)(x – β) = (x -【        】) (x -【       】)。

 

這就是x2 - x - 1的因式分解了。

它的首項係數和x2 - x - 1一樣，解也一樣。

 

簡單來說，要分解ax2 + bx + c，直接利用公式求得α, β。

分解的答案就是a(x – α)(x – β)。

 

其中α= [ -b + √(b2-4ac) ] / 2a, β=【              】。

 

別忘了前面要乘以a，這樣首項係數才會一樣。

因式分解的結果是

ax2 + bx + c = a(x – α)(x – β)

 

中間是等號，表示不只解相同，每項的係數都要相同。

 

 

我們先複習一元一次方程式中，根與係數的關係。

 

解ax + b = 0，x = ?

 

 

 

解ax + b = cx + d，x = ?

 

 

 

接下來，看一看二次方程中，根與係數的關係。

 

上頁我們得到了

ax2 + bx + c = a(x – α)(x – β)

 

其中a、b、c是二次、一次、常數項的係數，

α、β是方程式的兩解，稱為根。下式假定α較小。

α= [ -b - √(b2-4ac) ] / 2a

β= [ -b + √(b2-4ac) ] / 2a

 

比較常數項係數，我們會發現c = aαβ，也就是 αβ = c/a

比較一次項係數，我們會發現b = 【      】，也就是α+β = 【     】

 

αβ = c/a

α+β= -b/a

 

這兩條等式，就是「根與係數的關係」。

 

這兩條等式包含abcαβ共五個未知數。

任其中三數已知時，它就退化成二元方程組，用這兩條式子可以解出。
