我們先來看個例子：為什麼奇數加奇數是偶數？

奇數加偶數是【  】數？

偶數加偶數是【  】數？

 

奇數乘奇數是奇數？

奇數乘偶數是【  】數？

偶數乘偶數是【  】數？

 

這些事情需要硬記嗎？當然不需要。

 

因為，若引入符號，用n, m 等數代表未知的整數，那麼

奇數都可寫成2n + 1

偶數都可寫成2n

 

奇數加奇數，就可寫成(2n + 1) + (2m + 1)

 

 

 (2n + 1) + (2m + 1)

= 2n + 2m + 2

= 2(n + m + 1)

 

結果是2的整數倍，和2n屬於相同的型，所以是偶數。

 

2n就稱為偶數的「通式」；【     】則稱為奇數的「通式」。

多項式的研究起源於方程式求解的問題。

 

像x+2 = 5是一個方程式；左側的x+2就是一個多項式。

 

正規來說，多項式的定義，是由常數（一般的數字，如1, 2, 3, -5, √2, π）和變數（如x, y, z, a, b, c）透過「+」和「×」註[1]組合而成的式子。

 

例如：
2x + 6是2 × x + 6

5x3 - 7是5 × x × x × x + (-7)

4x3 – 2x - 8是【                    】

 

請注意，多項式的生成方式不包括「÷」，像、就不是多項式。

餘數是不是一定小於除數？【     】（是／否）

 

一般的除法其實代表了一個乘法與加法的等式：

被除數 = 除數 ×【       】+ 餘數    （其中餘數小於【      】）

 

所以凡是可做乘法與加法的東西，就可以做除法。多項式也不例外。

 

多項式的除法，在做高次的因式分解時，是相當常用的工具。

 

在多項式裡面，我們把除法等式

 

被除數  = 除數 × 商數 + 餘數           （其中餘數 小於 除數）

 

改為

 

被除式  = 除式 × 商式 + 餘式           （其中餘式 低於 除式）

 

為什麼改成「餘式低於除式」呢？因為多項式只能比高低啊。

 

和因倍數很相似，在多項式的除法中，如果餘式是零，

除式就稱為被除式的「倍式」；後者則是前者的「因式」。

在未知數的符號發明前，人們通常是用幾何圖形來證明公式，

用線段的長度來扮演現代x, y, z的角色，用面積來代表未知量的乘積。

 

舉例來說，以a, b代表兩個未知的長度，那麼：

 

 

a2,  ab,  b2都是代表面積；

a3,  a2b,  ab2,  b3如果畫出來，都是代表【       】。

 

以乘法公式(a+b)2 = a2 + 2ab + b2為例。

我們不難看出，(a+b)2 = a2 + 2ab + b2是一個談論面積的式子。

 

因為它的每個項，都是【   】次。

 

我們稱a2+2ab+b2為一個齊2次式，

更完整地說，它是一個「2元2次齊次3項式」。

 

因為幾何代數的公式都有幾何意義，一定是齊式，

如果背公式(a+b)2時，跑出一個a2b2，次數不對，那肯定是背錯了。

 

齊次性在物理、化學的公式上，也是很好的檢驗標準。
