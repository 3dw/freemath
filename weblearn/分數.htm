分數的變形

 

把分數改寫成同一族中，分子、分母比較小的，就叫作「約分」，反之就叫作「擴分」。

 

把寫成，叫作【  擴分  】；把寫成，叫作【  約分  】。

 

一般來說，約分是把分子分母同除以某數；擴分則是同乘以某數。

約分、擴分完全不影響分數的大小，只改變表面的形式。

 

擴約分在除法中對應的動作就是除數、被除數等比例縮放。

例如 2 ÷ 3 = 4 ÷ 6 = 6 ÷【  9  】。

由於分子、分母可以無限增大，大並不特別，反而簡單才是美。

前面提到的分數，分母都不是0，分子也很少出現0。

那是刻意避開的結果。

現在我們可以來考慮一下：如果0出現在分數中，會發生什麼事呢？

 

0 在分子時

0 在分子時，代表「拿了0份」，也就是「一份都沒拿」。

不管一份有多大，既然沒拿，結果當然都是「沒有」。

 

所以說，分子為0的分數，都等於0。

 

0 在分母時（情況一）

0在分母時，如果分子不是0，例如，代表多大的數呢？

從上往下唸，就是「3除以0」。

寫成原始的乘法，「3 ÷ 0 = 某數」，就代表「某數 × 0 = 3」。

所有數字乘以0都是【    】，沒有數字乘以0之後會變成3。

 

所以說，並不代表任何數。

同理，分母為0時，若分母不為0，則該分數無意義。

 

0 在分母時（情況二）

0在分母時，如果分子也是0，也就是，又是多大呢？

從上往下唸，就是「0除以0」。

寫成乘法，「0 ÷ 0 = 某數」，就代表「某數 × ０ = 【   】」。

 

但是，所有數字乘以0都是【     】。

所以可以是0,1,2,...或是...所有數字都可以。

 

所以，代表「所有數字」，是「數字的集合」，不是特定數字。
