(a+b)(a-b) = a2 –b2這條公式，其實有更深刻的意涵，

那就是：「所有周長相等的長方形中，正方形的面積最大。」

也就是說，若拿一條繩子，

圍出長方形的土地，面積愈大愈好，那麼「正方形」最好。

 

畫畫看，(a+b)(a-b)代表的長方形，周長是【        】。

畫畫看，a2代表的正方形，周長是【        】。兩者周長相等。

 

但是，在b不為零的情況下，

(a+b)(a-b) = a2 –b2告訴我們， (a+b)(a-b)  <  a2（正方形面積較大）

 

 

此外，平方差公式還可以解釋九九乘法表潛藏的秩序。像：

6×4 = 24，5×5 = 25，兩者差1；

7×5 = 35，6×6 = 36，兩者差1；

8×6 = 48，7×7= 49，兩者差【    】；

同樣把12拆成兩數。（不一定要12，你可以試試看別的偶數）

6×6 = 36，7×5 = 35，8×4 = 32，9×3 = 27，10×2 = 20

用最大的36去減它們，得到

36 - 36 = 0，36 - 35 = 1，36 - 32 = 4，36 - 27 = 9，36 - 20 = 16。

 

0, 1, 4, 9, 16，都是平方數。為什麼呢？想想看！

 

其實，和的完全平方、差的完全平方、平方差公式，從代數的觀點，都可以看成是(a+b)(c+d)的特例，只是有時c = a, d = -b等等。

 

如上圖，(a+b)(c+d) = 【    】+ 【    】+【    】+【    】

此式稱為乘法對加法的「分配律」。

 

有了分配律，我們就可用代入法導出公式。

什麼是代入法呢？

由於不管放什麼數，分配律都會成立，我們可以只看特例，

 

考慮c = a, d = b的特例，則

(a+b) (c+d) = ac + ad + bc + bd  就會代換成

(a+b) (a+b) = aa + ab + ba + bb  （c都換成【   】，d都換成【   】）

 

理項之後，就是【    】的完全平方公式。

 

考慮c = a, d = -b的特例，則

(a+b) (c+d) = ac + ad + bc + bd  就會代換成

(a+(-b)) (a+(-b)) = aa + a(-b) + (-b)a + (-b)(-b)

理項

(a-b) (a-b) = a2 - 2ab + (-b)(-b)

 

這很像【   】的完全平方公式，除了最後一項怪怪的。

其實，(-b) × (-b) = b2，所以上式就是【   】的完全平方公式。
