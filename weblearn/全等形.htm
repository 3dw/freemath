幾何圖形中，多邊形沒有彎曲，可說是較簡單的。

而多邊形都是由最簡單的【       】（填三角形或方形）所構成。

 

因此，要學習幾何證明，通常都是由三角證明入手。

很多證明的問題，都可以化為「證明兩個三角形全等」的問題。

 

兩個圖形全等的定義是，

它們可經由「平移」、【     】、【     】三種變換之後，彼此疊合。

（不多也不少，正好疊在一起）

 

 

全等三角形，就是可以經由平移、【        】、【        】之後，彼此疊合的兩個三角形。

 

如果兩個三角形三邊對應相等、三角對應相等，它們必然【     】。

 

前面我們利用平行線的性質，已經證明了許多和「角」有關的定理。

 

不過，要處理和「邊」有關的問題，需要新的工具，就是全等三角形。

 

三角形有三個邊（Side, 簡寫S）、三個角（Angle, 簡寫A）。

以下幾種條件，可保證兩個三角形全等：

 

1.    三邊（SSS）對應相等（公理3）

 

固定一已知邊，將另兩個【      】作為半徑畫圓，

可以唯一決定一個三角形。它也可能畫出鏡像，鏡像還是和本來全等。

 

2.    兩角一邊（AAS）或兩角一夾邊（ASA），對應相等。（公理4）

 

兩角相等，第三角也必【     】。所以考慮（ASA）的情況即可。

固定已知邊，用已知角向上延長，可以唯一決定一個的三角形。

 

3.    兩邊一夾角（SAS）對應相等（公理5）

 

用已知角出發，向左右伸出已知的邊長，決定【       】的三角形。

 

4. 直角帶兩邊（RHS）對應相等（公理6）

 

 

用已知直角出發，向上伸出已知的邊長，再用另個已知邊長畫【      】，可以【       】決定一個三角形。

 

為什麼跑出RHS來呢？它不是【      】的特例嗎？

下頁我們會討論一般的ASS情況，說明為什麼只有特例成立。

 

以上的說明，屬於概念性說明，相關的嚴謹證明較繁雜，這裡不細究，直接把SSS、AAS、ASA、SAS、RHS都當成公理，可以直接使用。

歐幾里德的經典之作：《幾何原本／Elements》

在應用全等三角形來作證明之前，先補充一下關於反例的知識。

 

證明一個「所有…都…」的命題，需要考慮所有可能的情況，

但是要推翻「所有…都…」的命題，只要舉出【    】個反例。

SSS、【      】和【       】、【       】和RHS五種。

定理9：

等腰三角形兩底角相等。

 

已知：AB = AC

求證：∠B = ∠C

 

證明：

1. 過A作BC的垂線，交BC於P點

2. △ABP和 △ACP中，

AP = AP（共邊）,

AB = AC（已知）,

∠APB = ∠APC = 90°

 

∴△ABP ≅ △ACP（RHS）

∴∠B = ∠C（對應角）

 

其中「≅」代表全等，也有人用「≡」來代表全等。

 

 

其實，這個定理有另一種證明方式，不需要輔助線：

 

已知：AB = 【      】

求證：【            】

 

證明：

△ABC 和 △ACB中，

AB = 【 AC  】（已知）,

AC = 【      】（已知）,

∠A = ∠A（共角）

 

∴△ABC ≅ △【      】（SAS）

∴∠B = ∠C（【         】）

 

 

秘訣在於，把△ABC和△ACB看做兩個不同的三角形。

 

定理10：

兩底角相等的三角形，必為等腰。

 

已知：【       】

求證：AB = AC

 

證明：

1. 過A作BC的垂線，交BC於P點

2. △ABP 和 △ACP中，

【       】=【       】（共邊）,

【       】=【       】（已知）,

【       】=【       】 = 90°

 

∴△ABP ≅ △ACP（【        】）

∴【           】（對應邊）

 

 

 

當然，如果使用秘訣，也可以不畫輔助線：

 

已知：【       】

求證：AB = AC

 

證明：

△ABC 和 △ACB中，

∠B = 【      】（【 已知 】）,

∠C = 【      】（【      】）,

∠A = ∠A（【      】）

 

∴△ABC ≅ △ACB（【       】）

∴【            】（對應邊）

三角形每邊中點的連線，將三角形分為彼此全等的四個小三角形。

AMc = BMc,  BMa = CMa,  CMb = AMb

AB // MaMb, BC // 【       】,  【     】 // McMa

不過，仔細觀察，就會發現已知條件中A、B、C的地位是對稱的。

也就是說，如果能證明△AMbMc ≅ △MaMcMb，

 

那麼用完全相同的步驟，

只是把A, B, C符號輪換（A→B→C→A），就可以證出

和

△CMaMb ≅ △【          】

 

這種符號上的對稱，稱為符號對稱，可以讓我們省很多力氣。

因為只要證出一部份的問題，就能證明全部的問題。

 

用部份代表整體的手法，

稱為「不失一般性」（Without Loss of Generality, 簡寫WLOG）手法。

∵A, B, C【     】對稱

∴WLOG，問題化簡為

只剩一個全等要證了，現在就來證吧！
